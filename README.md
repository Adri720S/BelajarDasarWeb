**Belajar Dasar Pemrograman Web**

Di sini saya membuat tampilan web dengan mengambil tema mobil sport: Koenigsegg Agera, langkah-langkahnya:
1. Buat code di html
2. Buat code di css dan sambungkan ke file html
3. Buat repository baru di github, sertakan file README.md
4. Upload file ke github 
5. Buat akun vercel app (jika tidak bisa pakai akun github, coba pakai email)
6. Buat project di vercel app
7. Koneksikan ke akun github
8. Pilih repositori yang kita inginkan
9. Selesai

Hasil proyek: https://belajar-dasar-web.vercel.app/
